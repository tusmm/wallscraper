# Wallscraper
Scrapes [wallhaven.cc](https://wallhaven.cc/) for its wallpapers given a search term

# Dependencies
Install these moduules using pip
```
pip install requests
pip install bs4
```

# Usage
For terminal, type `python wallscraper.py` to run program.\
